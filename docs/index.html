Description

This is a parallel coordinates explorer of Hyper-parameters tuned for a ResNet50 model on CIFAR-10 dataset. The Parallel Coordinates display the Accuracy, Loss and F1 Scores colored by Optimizer across 3 dimensions : Epochs, Batch Size and Learning Rate.


Exploration Guide

Hover over the viz, buttons and labels to view controls.

Brush the visualization to update other charts on this page.

To Brush: Drag vertically along an axis.
To Remove Brush: Tap the axis background.
To Reorder Axes: Drag an axis label horizontally.
To Remove Axis: Drag axis label to the left edge.
To Invert Axis: Tap on axis label.
To Reset Visualization: Refresh webpage.
